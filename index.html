2.2.3 常见Linux命令

1：查看当前目录下的文件：ls
2：查看当前目录下的文件，包括隐藏文件：ls -a
3：创建文件：touch 文件名
4：创建文件夹：mkdir 文件夹名

6：删除文件：rm -rf 文件名

2.3.1 提交到GitHub上

步骤1：git add：
可以用git add -A或者git add . 2个命令效果一样;
步骤2：git commit：
git commit -m “本次提交的修改的备注” ;
也可以直接用git commit -am “本次提交的修改的备注”来合并前2个步骤。
步骤3:git push:
第一次提交到本分支：git push origin master
origin是远程仓库的默认名称，master是我们的分支名称（主分支）
第二到n次提交到本分支可以直接用git push
提交到其他分支：git push origin b（会提交到b分支）

2.3.2 抓取远程仓库内容 :git pull
