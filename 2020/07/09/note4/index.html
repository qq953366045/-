4.3 分析用户行为
1：复杂统计出现最多的内容
GET /renders.test.com/g/agentsales/js/view.js HTTP/1.1
GET：当前的请求类型
/renders.test.com/g/agentsales/js/view.js：请求的网址
HTTP/1.1：请求的协议
2：统计网址请求次数前10名的命令：（请求的网址处于第8列）
cat cookie_log.log | awk ‘{print $8}’ | sort | uniq -c | sort -k1r | head -10
4.4 Shell脚本
1：脚本头：#！/bin/bash（必须放在脚本文件的第一行）
2：例子：#!/bin/bash
if [ -f ‘cookie_log.log’ ]; then
cat cookie_log.log | awk ‘{print $8}’ | sort | uniq -c | sort -k1r | head -10
fi
if [ -f ‘cookie_log.log’ ];是判断cookie_log这个文件是否存在，存在就执行下面的命令。
3：保存：Shell脚本保存一般已.sh作为后缀结尾如top10.sh
4：运行：sh top10.sh
